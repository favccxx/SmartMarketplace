# SmartMarketplace
SmartMarketplace是一个极简的前后端分离的商城管理系统，前端采用了易上手VUE2框架结合了webpack构建工具，支持权限管理、国际化以及动态侧边栏，可满足用户的日常操作需求。后端使用SpringBoot构建微服务，整合了Shiro、Swagger等组件，可以方便的进行权限管理和测试。

## 前后端分离利弊谈

**前后端分离的优点**

-	性能显著提升
-	术业有专攻
-	提升开发效率
-	前后端职责划分不清
-	解放前端
-	重用后端服务

**前后端分离的缺点**

-	小型项目不适用
-	人工成本
